# Karchopo · Desarrollador Web y Programador

Programador de aplicaciones web, interesado en el desarrollo de sistemas antiguos y creación de videojuegos.

---

### Proyectos Actuales

| Proyecto | Tecnologías | Descripción Detallada |
| :--- | :--- | :--- |
| **SIMS** | Laravel 12, Vue 3, PostgreSQL, Python | Desarrollo de una plataforma completa para la gestión de flotas de vehículos, reservas y telemetría. Incluye backend multitenant, interfaz reactiva e integración IoT con Raspberry Pi. |
| **Pico Launcher** | C++, libtwl, BlocksDS | Desarrollo continuo de nuevas funcionalidades, optimizaciones e implementaciones técnicas para este menú de Nintendo DS/DSi. |
| **StreamerLibrary** | Python, Flask, Twitch API | Herramienta de gestión de Twitch y biblioteca de VODs. Permite organizar y gestionar el contenido de las retransmisiones de forma eficiente. |
